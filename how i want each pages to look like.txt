1. System Actors (Role-Based Access)
Receptionist: Handles registration, check-ins, and assigning patients to doctors.

Nurse: Handles triage and records vitals.

Doctor: Reviews vitals, documents symptoms, requests labs, makes diagnoses, and prescribes medication/admission.

Laboratorist: Processes lab test requests and uploads results.

Pharmacist: Dispenses prescribed medications.

2. Patient Workflow & System States
Phase 1: Registration & Check-in (Receptionist)

Condition A (New Patient): Receptionist captures patient details and creates a new Patient Profile.

Condition B (Existing Patient): Receptionist searches via Patient ID or Name.

Action: Patient is checked into the system.

State Change: Patient status changes to Waiting for Triage and appears on the Nurse's Active Waiting List.

Phase 2: Triage (Nurse)

Action: Nurse selects patient from the waiting list, takes vitals (BP, temp, weight, etc.), and saves the data to the patient's current visit record.

State Change: Patient status changes to Waiting for Doctor Assignment. (Patient is routed back to the Receptionist queue).

Action: Receptionist assigns the patient to a specific, available Doctor.

Phase 3: Initial Consultation (Doctor)

Action: Doctor verifies patient identity, opens the Patient File, and reviews the nurse's recorded vitals.

Action: Doctor documents patient Symptoms.

Decision Point: Does the patient need lab tests?

If YES: Doctor creates a Lab Request. State changes to Waiting for Lab.

If NO: Skip Phase 4 and proceed directly to Phase 5.

Phase 4: Laboratory (Laboratorist) - Conditional

Action: Laboratorist verifies patient via Patient ID.

Action: Laboratorist conducts requested tests and inputs the Lab Results into the system.

State Change: Results are sent back to the Doctor. Patient state changes to Waiting for Doctor Review.

Phase 5: Diagnosis & Prescription (Doctor)

Action: Doctor reviews lab results (if applicable), makes a Clinical Diagnosis, and generates a Prescription.

Decision Point: Does the patient need to be admitted?

If YES: Doctor selects Admit Patient. State changes to Admitted. (Case stays open/transfers to inpatient ward).

If NO: Doctor finalizes the visit. State changes to Waiting for Pharmacy.

Phase 6: Pharmacy (Pharmacist) - Conditional

Action: Pharmacist accesses the patient's Prescription via the system.

Action: Pharmacist dispenses the drugs.

State Change: Visit is marked as Closed / Completed.
